# Retail Sales Performance Analysis (2010-2011): Leveraging Descriptive Analytics for Market Insights
## Project Objective
To conduct an Exploratory Data Analysis (EDA) on multi-country retail sales data to assess performance, identify key geographical markets, and provide a strategic basis for customer segmentation and inventory management.
